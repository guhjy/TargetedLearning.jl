computefluc(q::Qmodel{T<:FloatingPoint}, param::Parameter{T<:FloatingPoint}, gn1, A, Y)

Computes a new fluctuation for a (possible already fluctuated) initial Qmodel

Arguments

  • q - Qmodel
  • param - target parameter
  • gn1 - estimated propensity score
  • A - observed treatments
  • Y - observed outcomes

Keyword Arguments

  • weighted - A boolean, false by default If false, the "standard" fluctuation is performed, there the fluctuation covariate is divided by (g_n(A_imid W_i)), and weights are set to 1. If true, the fluctuation covariate is used directly, and weights are set to the reciprical of (g_n(A_imid W_i)) where (g_n(A_i mid W_i)) is computed as gn1[i] if A[i] is 1, or 1-gn1[i] otherwise.
